## RACI

راهنما:

- R = Responsible
- A = Accountable
- C = Consulted
- I = Informed

| فعالیت | محمدامین | محمدرضا | محدثه |
|---|---|---|---|
| تعریف scope | A/R | C | C |
| تحلیل ارزش اجتماعی | C | A/R | C |
| مدل ML | A/R | C | I |
| API | A/R | I | C |
| UI | C | I | A/R |
| KPI و trade-off | C | A/R | I |
| Risk Register | C | A/R | C |
| QA سناریوها | C | I | A/R |
| پوستر | C | C | A/R |
| ویدئوی نهایی | A/R | R | R |
| گزارش نهایی | A/R | C | R |
| Trello/Notion board | C | A/R | C |
| Knowledge Base | C | C | A/R |

## نکته دفاعی

این RACI نشان می‌دهد هر عضو نقش مشخص دارد، اما بار پیچیده فنی روی محمدامین است و نقش دو عضو دیگر طوری تعریف شده که در ارائه قابل توضیح و واقعی باشد.
