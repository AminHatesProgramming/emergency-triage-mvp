# یادداشت انتشار ۱.۰.۰

- انتشار نخست نسخه Android آفلاین امدادیار
- رابط فارسی و راست‌به‌چپ برای موبایل و دسکتاپ
- مدل مرورگری نسخه ۷ بدون وابستگی به سرور
- پشتیبانی از ارزیابی با اطلاعات ناقص
- تفکیک سطح فوریت از درصد خام مدل
- لایه ایمنی سن‌محور برای ضربان قلب و سایر علائم حیاتی شدید
- هشدار تکرار اندازه‌گیری برای داده‌های بسیار پرت
- نمایش روشن «فوری»، «سریع» و «مسیر معمول»
- هشدار استفاده مسئولانه و تماس مستقیم با اورژانس ۱۱۵
- بدون ثبت‌نام، تبلیغ، خرید درون‌برنامه‌ای یا مجوز حساس

این نسخه آزمایشی و آموزشی است و هنوز اعتبارسنجی بالینی نشده است.
