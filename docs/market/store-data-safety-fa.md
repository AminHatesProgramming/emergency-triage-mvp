# پاسخ پیشنهادی Data Safety و حریم خصوصی

پاسخ‌ها براساس نسخه Android 1.0.0 و manifest نهایی تنظیم شده‌اند. متن دقیق سؤال‌ها در پنل هر مارکت ممکن است متفاوت باشد.

| پرسش | پاسخ نسخه ۱.۰.۰ |
|---|---|
| آیا برنامه داده کاربر جمع‌آوری می‌کند؟ | خیر؛ داده فرم برای ارزیابی محلی پردازش می‌شود و به توسعه‌دهنده ارسال نمی‌شود. |
| آیا داده‌ای با شخص ثالث به اشتراک گذاشته می‌شود؟ | خیر. |
| آیا داده سلامت وارد می‌شود؟ | بله، کاربر ممکن است علائم حیاتی و سابقه بالینی وارد کند؛ این داده فقط در حافظه اجرای محلی پردازش می‌شود و «جمع‌آوری» نمی‌شود. |
| آیا برنامه مجوز اینترنت دارد؟ | خیر. |
| آیا حساب کاربری لازم است؟ | خیر. |
| آیا تبلیغ یا رهگیری وجود دارد؟ | خیر. |
| آیا خرید درون‌برنامه‌ای وجود دارد؟ | خیر. |
| آیا موقعیت، مخاطبان، دوربین، میکروفن یا فایل‌ها خوانده می‌شوند؟ | خیر. |
| چه چیزی در دستگاه ذخیره می‌شود؟ | فقط پذیرش یک‌باره هشدار استفاده مسئولانه در `localStorage`؛ داده بیمار ذخیره نمی‌شود. |
| کاربر چگونه داده را حذف می‌کند؟ | با دکمه پاک‌کردن فرم؛ داده موقت با بستن/بازنشانی برنامه از بین می‌رود. پذیرش هشدار با پاک‌کردن داده برنامه یا حذف برنامه پاک می‌شود. |
| آیا داده در انتقال رمزگذاری می‌شود؟ | انتقال داده بیمار وجود ندارد. |
| مخاطب کودک است؟ | خیر؛ برنامه برای نمایش آموزشی و ارزیابی توسط کاربران بزرگسال طراحی شده است. |
