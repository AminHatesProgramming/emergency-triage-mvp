## Definition of Done

- task assignee داشته باشد.
- زمان واقعی یا تخمینی ثبت شده باشد.
- خروجی قابل مشاهده در repository یا مستندات داشته باشد.
- برای deliverableهای اصلی، حداقل یک نفر دیگر review کند.
