| مستندسازی بخشی از رویدادها با تأخیر | بازسازی فقط از شواهد Git و مشخص‌کردن موارد فاقد تاریخ | کاهش ادعای بدون شاهد؛ نیاز به تکمیل پلتفرم |

## قواعد توقف ایمن

- خروجی سامانه نباید بدون بازبینی متخصص مبنای تصمیم درمانی قرار گیرد.
- اگر داده‌های حیاتی کلیدی موجود نیست، رابط باید کمبود داده را آشکار کند.
- اگر پایلوت محلی افت Recall یا نابرابری جدی نشان دهد، انتشار عملیاتی متوقف و مدل بازبینی می‌شود.
- تا پیش از مجوز و اعتبارسنجی، هیچ عبارت «سامانه بالینی»، «همکاری رسمی» یا «تأیید بیمارستان» استفاده نمی‌شود.
