# حاکمیت استفاده از ابزارهای هوشمند و کنترل انسانی

## اصل شفافیت

در امداد یار از ابزارهای هوشمند توسعه و تحلیل برای افزایش سرعت کدنویسی، بازبینی مستندات، پیشنهاد سناریوهای آزمون و استخراج شواهد استفاده شد. تصمیم نهایی، بررسی صحت فنی، پذیرش تغییر و مسئولیت اخلاقی در همه مراحل بر عهده تیم باقی ماند.

## دامنه استفاده و کنترل

| حوزه | نوع استفاده | کنترل انسانی |
|---|---|---|
| توسعه نرم‌افزار | پیش‌نویس تغییرات frontend، backend و build | بررسی diff، اجرای آزمون و پذیرش آگاهانه تغییر |
| تحلیل مدل | پیشنهاد بررسی خطا، آستانه و سناریوی مرزی | اتکا فقط به داده held-out، گزارش v7 و آزمون بازتولیدپذیر |
| مستندسازی | ساختاردهی گزارش، backlog و بسته تحویل | تطبیق با rubric، Git، Jira، Notion و شواهد واقعی |
| کنترل کیفیت | پیشنهاد test case و کنترل ناسازگاری | اجرای ۱۱۸ تست Python و ۱۱۶۷ سناریوی مرورگر/API |

## تصمیم‌های محفوظ برای تیم

- انتخاب مسئله سلامت‌محور و ارزش انسانی پروژه.
- تعریف سامانه به‌عنوان پشتیبان تصمیم، نه جایگزین متخصص.
- حذف ویژگی‌های post-triage و کنترل data leakage.
- اولویت دادن به Recall همراه با گزارش شفاف Precision و FPR.
- انتخاب طراحی فارسی، راست‌به‌چپ، mobile-first و انتشار آفلاین Android.
- تفکیک بازخورد کاربردپذیری از اعتبارسنجی بالینی.
- باز نگه داشتن تست دستگاه واقعی و پایلوت درمانی تا زمان دریافت شاهد.

## کنترل کیفیت و رد خروجی نامناسب

- تغییر فنی بدون diff و آزمون پذیرفته نشد.
- متریک‌ها فقط از فایل‌های مدل v7 و گزارش release استخراج شدند.
- داده‌های آزمایشی از ۹ بازخورد تأییدشده پرستاران تریاژ جدا ماندند.
- آستانه عمومی برای همه ورودی‌های ناقص رد شد؛ فقط الگوهای دارای ارزیابی held-out استفاده شدند.
- متن‌ها و اجزای غیرضروری از رابط محصول حذف شدند تا مسیر اصلی ساده بماند.
- هیچ نصب فیزیکی، همکاری بیمارستانی یا اعتبارسنجی بالینی بدون شاهد Done اعلام نشد.

## ملاحظات اخلاقی

- خروجی سامانه تشخیص پزشکی یا دستور درمان نیست.
- خطای False Negative برای بیمار پرخطر و FPR مرتبط با فشار منابع باید همزمان دیده شوند.
- داده بیمار واقعی فقط با مجوز، محرمانگی و حذف اطلاعات هویتی قابل استفاده است.
- هرگونه استفاده عملی نیازمند اعتبارسنجی بالینی و تأیید مرکز درمانی است.
- مسئولیت تفسیر و تصمیم نهایی با متخصص انسانی و پروتکل رسمی مرکز است.
