## نقاط عطف قابل نمایش روی پوستر

1. انتخاب مسئله سلامت و بحران
2. ساخت مدل اولیه
3. کنترل leakage
4. ساخت API
5. ساخت UI
6. آموزش مدل v6
7. آماده‌سازی پوستر و گزارش نهایی

## دستاوردهای اخیر

- مدل v6 با AUC برابر 0.8947 و Recall برابر 0.9241
- API قابل demo با `/predict`
- UI فارسی mobile-first
- پشتیبانی از ورودی ناقص
- مستندات کامل برای نمره‌دهی پوستر و تحویل نهایی
